# 総合想定評価フォーム

救急救命士向け想定訓練の評価を、スマホ・タブレットからその場で入力・送信できる HTML アプリです。
タイマーで活動時間を計測し、大項目ごとに 3 段階で評価して Google スプレッドシートへ記録します。

- **運用主体**：駿東伊豆消防本部
- **バージョン**：Ver.2.0（2026年7月）
- **本体ファイル**：`index.html`（QR コード・配信 URL の参照先）

---

## 主な機能

- **接触〜病着タイマー**
  接触 / 離脱 / 現場出発 / 病着 の 4 ポイントをワンタップで記録。
  「接触〜離脱」「離脱〜現場出発」の経過時間を自動計算して表示します。

- **基本情報の入力**
  - 実施日（日付）
  - 実施隊（**選択式**：沼津北１・沼津北２・原・沼津南・静浦・西浦・戸田・清水町１・清水町２・田方北・田方中１・田方中２・田方南・田方西・伊東・八幡野・宇佐美・吉田・東伊豆 の 18 隊）
  - 部（**選択式**：1部 / 2部 / 3部）
  - 隊長・隊員1・隊員2・評価者（氏名）

- **大項目評価（3 段階）**
  以下 6 つの大項目を **適切 / 不適切 / 該当なし** で評価します。
  各項目には判断の目安となる観察ポイントを参考表示（採点対象外）。

  | 項目 | 内容 |
  |------|------|
  | 🏃 活動全般 | プロトコール遵守・現場滞在時間・方針共有・温かい人間愛と思いやりの心をもった対応 |
  | 🔍 初期評価・観察 | ABCD 評価・事案に応じた観察・対応の適切性 |
  | 💓 ハイパフォーマンスCPR | 人工呼吸・胸骨圧迫・パルス／モニターチェック |
  | 💉 特定行為等 | 手技・実施順位・滞在時間・IC・医師確認 |
  | 📞 病院連絡 | キーワード・MIST・指示要請・プレゼン・資格説明 |
  | 🤝 説明と同意 | 活動前の説明同意・活動後の説明 |

- **総括コメント**
  自由記述で総合的な評価コメントを入力。

- **送信**
  入力内容を Google Apps Script（GAS）経由でスプレッドシートへ送信します。

## 画面デザイン

- ヘッダー帯は救急イメージの青系（`#1565C0`）、文字は白。
- スマホ表示前提のレスポンシブレイアウト（最大幅 600px）。
- 最下部フッターに「駿東伊豆消防本部 / Ver.2.0 / 2026年7月」を横並び表示。

---

## 使い方

1. `index.html` をスマホ／タブレット／PC のブラウザで開く
2. 訓練の進行に合わせてタイマーボタンをタップ
3. 基本情報・各項目の評価・コメントを入力
4. 「📤 評価結果を送信する」をタップ
   （**実施日・実施隊・部** は必須。未入力だと送信できません）

---

## 送信先の設定

送信データは Google Apps Script の Web アプリ経由でスプレッドシートに記録されます。
送信先を変更する場合は、HTML 内の以下の定数を書き換えてください。

```js
const GAS_URL = 'https://script.google.com/macros/s/.../exec';
```

送信は `mode: 'no-cors'` で行うため、ブラウザ側でレスポンスは読めませんが、GAS 側には記録されます。

---

## 送信データ（payload）

| キー | 内容 |
|------|------|
| `submittedAt` | 送信日時 |
| `date` | 実施日 |
| `unit` | 実施隊（選択した隊名） |
| `bu` | 部（1部／2部／3部） |
| `leader` / `member1` / `member2` | 隊長・隊員1・隊員2 |
| `evaluator` | 評価者 |
| `items` | 各大項目の評価値（適切／不適切／該当なし／未評価） |
| `timerContact` / `timerDepart` / `timerSceneOut` / `timerHospital` | 各タイマー時刻 |
| `elapsed1` / `elapsed2` | 経過時間 |
| `comment` | 総括コメント |

---

## 技術メモ

- 単一ファイルの HTML（外部ライブラリ・ビルド不要）
- 評価項目は HTML 内の `SECTIONS` 配列で定義。項目の追加・変更はこの配列を編集
- 実施隊・部の選択肢は `基本情報` の `<select>` を編集して増減
